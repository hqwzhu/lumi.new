为什么还不能公开下载

当前 DMG 是未签名内部测试版。面向普通 Mac 用户发布前，需要配置 Apple Developer 证书、完成代码签名和 notarization，否则 macOS Gatekeeper 可能阻止启动。

下一步

  1. 在 Mac 真机安装并完成首次启动验收。
  2. 配置 GitHub Actions Apple 签名密钥。
  3. 重新构建 signed macOS release kit。
  4. 测试通过后发布正式 macOS GitHub Release。
